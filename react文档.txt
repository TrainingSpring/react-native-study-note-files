安装react脚手架：
      npm：
          npm install -g create-react-app


查看是否安装成功：
	create-react-app --version //查询版本号 	


将npm的register改为cnpm：
	npm config set registry https://registry.npm.taobao.org

安装yarn  （可选）：
	npm install -g yarn

安装 react-router-dom     	  axios              	less	 less-loader

利用脚手架创建项目：
	create-react-app  newProjectName
